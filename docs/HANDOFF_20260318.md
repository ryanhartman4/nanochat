## Key Research Findings

### The Depth Signal Dilution Thesis
Three independent papers (AttnRes, DCA, MoDA) converge on the same diagnosis: standard residual connections with PreNorm cause hidden-state magnitudes to grow as O(L) with depth, progressively diluting each layer's contribution. This is the primary bottleneck for nanochat's d24-d26 models.

### Existing Mechanism in nanochat
nanochat already has x0 re-injection via `x0_lambdas` (line 452: `x = resid_lambdas[i] * x + x0_lambdas[i] * x0`). The current prescribed decay (0.20→0.05) may be suboptimal. The flat 1.0 experiment lets the optimizer discover the ideal curve.

### Recommended Implementation Order
1. **x0 re-injection** (in progress) — trivial, tests whether better scalars alone help
2. **AttnRes** — 1 day, replaces fixed x0 blend with softmax attention over all prior layer outputs
3. **DCA** — 1-2 days, composes Q/K/V from historical layer outputs (complementary with AttnRes)
4. **MoDA** — 2-3 days, adds depth KV inside attention (needs Triton kernel)

AttnRes and DCA are **complementary** — they modify different parts of the forward pass. Both could be stacked.
